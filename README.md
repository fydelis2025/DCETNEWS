# DCETNEWS - Portal de Informações Acadêmicas

## 📌 Sobre o Projeto
O **DCETNEWS** nasceu da ideia de centralizar e modernizar o fluxo de informações do Departamento de Ciências Exatas e da Terra (DCET) da UNEB. O objetivo principal foi criar uma interface ágil e acessível para a comunidade acadêmica.

> **Nota de Desenvolvimento:** Este repositório representa um marco na minha transição de projetos privados para o compartilhamento público. Embora o sistema não tenha atingido sua implementação completa conforme o planejado originalmente, ele serve como um registro do meu processo criativo e das tecnologias que domino.

---

## 👨‍💻 O Projeto como Biografia Técnica
Este repositório é mais do que código; é um reflexo da minha experiência em **Software Engineering** e **Cybersecurity**. Ao longo do desenvolvimento, apliquei conceitos de:

* **Modernização de Interface:** Uso de tecnologias como HTML5 e CSS3 (com foco em performance para hardware limitado).
* **Estruturação de Dados:** Organização de fluxos de notícias e comunicados.
* **Segurança por Design:** Mentalidade de proteção de dados e integridade da informação desde o rascunho inicial.

## 📱 Expansão Mobile (Android)
O DCETNEWS foi planejado para evoluir além do navegador. A versão Android foca em:
* **Leveza:** Consumo mínimo de RAM (foco em dispositivos de entrada).
* **Notificações Push:** Alertas em tempo real sobre editais e notícias do DCET.
* **Segurança:** Implementação de protocolos HTTPS e proteção contra ataques de intercepção de dados (MitM).
